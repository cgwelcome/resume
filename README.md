# Charles Gao — Resume

LaTeX source for my resume.

## Prerequisites

Install BasicTeX and required packages:

```bash
brew install --cask basictex
sudo tlmgr install titlesec
sudo tlmgr install enumitem
sudo tlmgr install fontspec fira
```

## Build

```bash
lualatex CharlesGaoResume.tex
```

Output: `CharlesGaoResume.pdf`
